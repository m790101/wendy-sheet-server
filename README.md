# Inventory-helper

A simple inventory management tool built with Node.js (Express) and MongoDB.

## Features

- Users can count and manage consumable items
- Add, delete, and update items in the inventory
- Export inventory data to Excel files

## Tech Stack

- Node.js (Express)
- MongoDB
